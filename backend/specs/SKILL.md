# Sentinel-01 Skills
## AETHEL Foundation

### Core Skills

#### 1. Market Analysis
- Real-time price monitoring
- Volatility calculation
- Trend detection (direction + strength)
- Liquidity assessment
- Anomaly detection

#### 2. Regime Classification
- **NORMAL**: Standard market conditions
- **VOLATILE**: Elevated volatility
- **CRISIS**: Extreme conditions
- **UNKNOWN**: Insufficient data

#### 3. Risk Assessment
- Pre-trade checklist evaluation
- Portfolio exposure analysis
- Drawdown monitoring
- Position sizing

#### 4. Policy Enforcement
- Regime-based action filtering
- Constitutional limit enforcement
- Governance directive compliance

#### 5. Trade Execution
- TradeIntent construction
- EIP-712 signing (future)
- Simulation execution
- Result tracking

#### 6. Audit Trail
- ValidationArtifact generation
- Hash computation
- Reputation metric updates
- On-chain publishing (future)

### Skill Boundaries

The agent **CAN**:
- Analyze market data
- Propose trades within policy
- Execute approved trades
- Record all decisions

The agent **CANNOT**:
- Violate risk limits
- Override governance
- Self-modify parameters
- Act without audit trail
